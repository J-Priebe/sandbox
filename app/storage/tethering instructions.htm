Overview

Depending on the wireless service provider, a BlackBerry smartphone can be connected to a computer with a USB cable and be used as a modem to connect to the Internet when the computer might not otherwise have access to the Internet.

One BlackBerry smartphone can be used as a modem at a single time. If the BlackBerry smartphone is associated with a BlackBerry Enterprise Server, contact the BlackBerry Enterprise Server administrator to make sure that the appropriate IT policy is turned on to use this feature.

Set up mobile Internet settings

Before a BlackBerry smartphone can be used as a modem, set up the mobile Internet settings. Select or create a profile that contains the information required by the wireless service provider to permit the Internet connection. Some profiles have been provided, but new profiles can also be created by completing the following steps:

  1. Connect the BlackBerry smartphone to the computer.
  2. Launch the BlackBerry Desktop Software and click Tools > Mobile Internet settings.
  3. Do one of the following:
    • Click the profile for the wireless service provider in the Profile drop-down list. In the Profiles Settings section, type any required information that is not already included in the profile. For more information, contact the wireless service provider.
    • Click Add custom profile if the wireless service provider does not appear in the Profile drop-down list. Type a name for the profile. Contact the wireless service provider to determine the information that is required for the Profiles Settings section. Required information can vary for each wireless service provider.
  4. Click OK.

Connect to the Internet using the BlackBerry smartphone as a modem

To perform this task, set up mobile Internet settings to create a connection profile. Depending on the wireless service plan, extra fees might apply.

  1. Connect the BlackBerry smartphone to the computer.
  2. In the BlackBerry Desktop Software, click Tools > Start Mobile Internet.
  3. Click Connect.

Note: A screen will appear displaying the connection time and amount of data downloaded.

To end the connection, click the Internet connection status. If the BlackBerry Desktop Software is already closed, click on the Dial-Up icon in the notification area of the Windows task bar.

Additional Information

Note: If the Start Mobile Internet and the Mobile Internet Settings options are not displayed under the Tools menu per the steps listed above, see KB24081.

Note: Some carriers have multiple access points. Please contact the carrier to obtain the correct access point if configuring a custom profile.
